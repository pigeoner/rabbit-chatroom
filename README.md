## 项目启动

### 1. 客户端
#### 安装依赖

```bash
$ npm install
```

#### 开发环境启动

```bash
$ npm run dev

# For macOS
$ npm run dev:mac
```

#### 构建

```bash
# For windows
$ npm run build:win

# For macOS
$ npm run build:mac

# For Linux
$ npm run build:linux
```
